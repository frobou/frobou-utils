# Frobou Utils

Utilitários de uso geral

### Contém

- Funções de uso geral
- Um ipcalc meia boca
- Um tail
- Um contador de tempo
- Funções de Cors
- Funções de getops

### Instalação

Instalação pelo composer

```
composer require frobou/frobou-utils
```
